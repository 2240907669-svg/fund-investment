# 免费数据源与降级规则

## 来源优先级

1. 基金管理人产品页、招募说明书、基金合同和公告：费率、份额、申赎状态及限额的权威来源。
2. 证监会及基金业协会：法规和行业规则。
3. 无需登录的公开基金聚合页面：净值历史和候选发现。当前适配器使用天天基金公开页面数据并标记为 `public-aggregator`。
4. 本地 CSV：网络不可用或页面结构变化时的审计后备。

## 网络纪律

- 只访问公开、无需登录的 URL；设置清晰的 User-Agent，串行请求并限制频率。
- 不规避验证码、登录、付费墙、地域限制或封禁，不调用销售交易接口。
- 保存来源 URL、抓取时间和原始响应摘要；解析失败时保留旧数据并报告失败。
- 聚合源与基金公司披露冲突时，以基金公司文件为准并暂停相关建议。

首次加入基金、费率文件更新、申赎状态变化或验证日期超过 90 天时重新核验。无法获得产品文件时把状态设为 `unknown`。
